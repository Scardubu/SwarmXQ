# packages that require precise version pinning due to C-API ABI constraints.

# Bounds explained:
#   numpy <2.0     — faiss-cpu 1.x links against the numpy 1.x C API
#   scikit <1.6    — 1.6 introduced breaking changes to the embedding pipeline
#   faiss-cpu <2.0 — major version boundary; 2.x API is incompatible
#   sentence-transformers <3.0 — 3.x deprecates SentenceTransformer.encode signature
#   sqlite-vec <0.2 — experimental extension; minor bump may shift symbol names
#   redis <6.0     — 6.x redis-py dropped compat shims used by BullMQ bridge
#   requests <3.0  — 3.x drops Python 3.11 transport-level fallbacks
#
# To upgrade: bump the version, run `make test`, then commit.
# Never leave bare package names — implicit latest will silently break builds.
